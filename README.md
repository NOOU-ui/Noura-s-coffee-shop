## Noura's - Menu Digital Artisanal

Ce projet est une vitrine interactive pour un établissement fusionnant le concept de "Coffee Shop" moderne avec les saveurs traditionnelles du Maroc. 

##  L'Identité du Projet
L'objectif était de créer une interface qui respecte les codes du web design actuel tout en intégrant une identité culturelle forte. On y retrouve des spécialités comme  le thé à la menthe traditionnel et des pâtisseries fines.
